Board Game Tools

Created by Ralf Tischer, Wuppertal, Germany in 2022 as the final project for the CS50 course.

It provides an open and modular architecture to be extended later by further tools.

The first module draws a start player: All players lay a finger on the touch screen. Each player is represented by a color. A timer starts and when expired, the color of the start player is randomly drawn.
